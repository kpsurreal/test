JS端

所有返回处理结果的JS函数都必须带参数callerId

函数: log(msg: String) -> Void

函数: computeAddResult(callerId: String, left: Int, right: Int) -> Void
说明: 处理结果通过Native端的OnResultCallback方法返回。

Native端

Native端需要将提供函数调用的对象在WebView中注入js端的window中，对象命名为common 函数: log(msg: String) -> Void

函数: computeAddResult(left: Int, right: Int) -> Int

函数: onResultCallback(callerId: String, receivedValue: String) -> Void
说明: js端函数的回调结果将在次方法中回调
